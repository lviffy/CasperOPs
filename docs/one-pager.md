# CasperOPs

## No-Code Trustworthy AI Agents on Casper

---

**What** | Build autonomous DeFi & RWA agents with a visual drag-and-drop builder. No coding required.
**How** | 19 Casper-native tools + 6 Odra smart contracts + x402 pay-per-call protocol
**Why** | Every agent action is cryptographically signed, recorded on-chain, and verifiable. Escrow holds funds until work completes.

---

### Key Features
- Visual workflow builder (React Flow) with 4 pre-built templates
- x402 payment protocol — pay-per-call in CSPR via HTTP 402
- Marketplace with reputation scores + escrow hiring
- Telegram bot for on-the-go agent management
- MCP server for LangGraph / CrewAI integration
- CSPR.click wallet — no seed phrases, no EVM bridge
